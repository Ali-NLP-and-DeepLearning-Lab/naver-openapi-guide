# 용어(Glossary)

<a name="directives"></a>
## 지시자(Directives)

Clova Interface Connect 에서 Client 로 전달되어지는 지시자 명령

<a name="events"></a>
## 이벤트(Events)

Client 에서 Clova Interface Connect 로 전달되어지는 요청

<a name="downstream"></a>
## 다운스트림(Downstream)

Clova Interface Connect 에서 Client 로 Directive 를 전달하기 위해 연결하는 Channel
